HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user.
  • This is relatively straightforward so far.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Style can be added inline, but it is better to link to a .css file.
  • a * applies to all elements, a . applies to class elements, other elements are called out individually.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branchName: creates a new branch and switches to it
  • git checkout main: switches to main branch
  • git pull origin main: main sure branch is up to date before submitting a pull request
  • git add -A: commit all changes to the branch
  • git commit -m "": commit and add message

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Arrays start at index 0 and are iterated thru in a standard C++kind of way
